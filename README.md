## Installation

1. Download and install [Node.js](http://nodejs.org/) (that will install npm as well)
2. CD to client folder.
3. Install the dependancies.<br />
   ```
   npm install
   ```
4. You are done for the setup, run the app using
   ```
   yarn start
   ```
